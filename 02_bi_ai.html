소개

저는 5년차 백엔드 개발자로서, 주로 Java와 Spring Boot를 사용하여 확장 가능하고 효율적인 시스템을 구축해왔습니다. 클라우드 기술과 DevOps에 관심이 많아 AWS, Docker, Kubernetes 등을 학습하고 있습니다. 새로운 기술을 배우고 적용하는 것을 좋아하며, 동료들과의 협업을 중요하게 생각합니다.

기술스택

Java Spring Boot JPA MySQL AWS Docker Git
